How to run the Blood Bank & Donor Management System Using PHP and MySQL

1. Download the zip file

2. Extract the file and copy Red_Wings folder

3.Paste inside root directory(for xampp xampp/htdocs, for wamp wamp/www, for lamp var/www/HTML)

4.Open PHPMyAdmin (http://localhost/phpmyadmin)

5. Create a Database with the name login

6. Import login.sql file(given inside the zip package in the SQL file folder)

7. Run the script http://localhost/Red_Wings/homepage.php(for home page)

8.Run the script http://localhost/Red_Wings/index2.php(for admin page)
